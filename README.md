# Wascraper
Repo ini berisikan javascript untuk mengekstrak nomor dari grup whatsapp 

# Cara Penggunaan
- Buka https://web.whatsapp.com/
- Tekan F12
- Masuk ke console
- Pastekan script yang sudah dibuatkan
- Maka akan terdownload CSV nya
